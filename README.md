# 📊 Superstore Sales & Profit Analysis Dashboard

## 🚀 Project Overview

This project presents an interactive Tableau dashboard built using the Superstore dataset.

## 📌 Key Features

* Total Sales, Profit, Orders, Profit Margin
* Region-wise sales analysis
* Category & Sub-category insights
* Top customers analysis

## 📊 Key Insights

* Technology category has highest profit
* Furniture has lower profitability
* West region has highest sales

## 🛠 Tools Used

* Tableau
* Excel
